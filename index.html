Botception

A bot for creating bots in flock

Inception of Bots

Botception provides an ease for creating bots. Automate all your redundant tasks without the need to code them! We think a lot of your tasks and bots can be created using simple easy to use bot

Supported Commands

Basic Ones

  • create-bot: Create a new bot
  • my-bots: get a list of your bots
  • help bot: get functionalities that can be added to a bot
  • help step: learn how to create atomic tasks that your bot can run
  • help task: learn how to create a task sub tasks
  • add: Add a data value to a bot
  • delete: Delete a data value from a bot
  • get: Get a data value from a bot
  • get-all: Get all data values of a bot

For creating workflows

  • create-step: create an atomic task that can be combined with other tasks
  • create-task: create a task to be executed
  • pipe: create a task as a seriers of steps
  • repeat: repeat task on regular interval of time
